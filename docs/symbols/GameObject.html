GameObject(I)
The default base class for all objects you can add to the engine. Events: create When object is created for the first time. step Triggered every update step. destroy Triggered when object is destroyed. Use the destroy event to add particle effects, play sounds, etc. remove Triggered when the object is removed from the engine. Use the remove event to handle any clean up.
